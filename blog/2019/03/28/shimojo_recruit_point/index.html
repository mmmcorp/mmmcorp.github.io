一緒に働きたい人 (下條編)

Tweet

こんばんみ、下條です。

弊社では全員が採用プロセスに関わっています。その一環として各メンバーが思いをブログに書いていたりもしています。

  • 岡本: 一緒に働きたい人
  • 内山: 採用選考で見ているポイント

今回は、私の視点から一緒に働きたい人について重要な要素について書いてみたいと思います。

なお、これから弊社への応募を考えている方におかれましては、これを見て何らかの対策をしてほしいという観点ではなく、あくまでマッチングしそうかという点でご一読いただければ幸いです。

一番重要な要素、それは、、、

一緒に楽しく働けそうか

です。

弊社の佐々木がよく言っているのですが、仕事は人生の中で大きな割合を占めるものだから楽しんで仕事をしたいし、社員にも楽しんで仕事をしてもらいたいと思っている、と。

そして私自身も仕事は楽しくやりたいと強く思っています。仕事ですので当然ながら大変なことや辛いこともありますが、それも含めて楽しくやりたいものです。

それではどういうときに仕事の楽しさを感じるかと考えてみると、

  • お客様の役に立ち、お客様の笑顔をみることができたとき
  • 困難な課題を克服して成長できたと思ったとき
  • 知的好奇心が満たされたとき

このあたりでしょうか。そして楽しく仕事をする上で重要なのは、もちろん仕事の内容も重要なのですが、一緒に働く仲間もそれ以上に重要であり、上記のような楽しさを分かち合える方と一緒に働きたいと思っています。

  • お客様のこと・チームのことを考えて行動できる人
  • 向上心がある人
  • 常に考えている人
  • 挑戦を恐れない人

こういった方は一緒に仕事をしていて刺激的で楽しいです。

また、これらは弊社の行動指針とも合致していると考えており、上記の要素を満たしている方はMMMで活躍・楽しく働けると考えています。逆に満たしていない部分があるなーという方はMMMで働くことは楽しくないかもしれません。

手前味噌的ではありますが、弊社ではメンバー皆が自分のやるべきことを考えながら自分で動いており、技術的にも精鋭ぞろい。
そんなメンバー達に囲まれながらMMMで働くことは決して楽ではありません。

しかし決して
楽でない = 楽しくない ではありません。
ここは非常に重要なので強調いたします。

先に挙げた要素を満たしている方にとっては非常に楽しく働ける会社だと思います。

ですので、

  • お客様のこと・チームのことを考えて行動できる人
  • 向上心がある人
  • 常に考えている人
  • 挑戦を恐れない人

こういった方とぜひ一緒に働きたいと思っていますし、自信を持ってMMMにご応募いただきたいと思います。

一緒に楽しく成長しあえる仲間を募集しています。新たな出会いをお待ちしています！
