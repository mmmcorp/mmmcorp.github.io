[AWS Lambda 入門] 簡単な関数をサポートされている全言語で紹介

Tweet

AWS Lambdaでサポートされている言語

2020年3月29日現在、AWS Lambdaでは 最新のサポート対象 として以下の言語が挙がっています。

  • .NET Core 2.1(C#/PowerShell)
  • Go 1.x
  • Java 11
  • Node.js 12.x
  • Python 3.8
  • Ruby 2.7

※上記のうち、AWSコンソール上で直接コードを記述できるのはNode.js, Python, Rubyのみ

今回は、AWS Lambdaでサポートされている全言語で関数を実装していきます。

それでは1つずつ関数を作成してAWS Lambdaにデプロイしていきましょう。
