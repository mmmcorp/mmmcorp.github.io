クラウド業界ニュースまとめvol.1 by MMM

Tweet

こんにちは、MMMです。

最近は少しずつ暖かくなり、春が近づいてきてきるように感じます。花粉症の方には辛い季節がやってきたかもしれませんね。

本日より、MMMが注目したクラウド業界の最新ニュースを定期的に紹介してきたいと思います。クラウド活用の専門家として、クラウド技術や働き方、業界のビジネス動向などの情報を幅広くお届けしますので、参考にして頂ければ幸いです。

クラウド技術

Savings Plan Update: Save Up to 17% On Your Lambda Workloads | AWS News Blog(2020/2/20)

AWS Lambdaに対して割引プランの適用が開始され、最大で17%のコスト節約ができるようになりました。AWSのコストを可視化するAWSコストエクスプローラーでは、割引プランの推奨をするとき、Lambdaの利用を考慮に入れるようになっています。

The State of Serverless | Datadog (2020/2/20)

DatadogによるLambdaの利用状況が公開されていました。Lambdaで使われている言語がPythonとNode.jsでほぼほぼ占められている点が印象的です。

Supercharge your command line experience: GitHub CLI is now in beta - The GitHub Blog(2020/2/12)

コマンドラインからGitHubを操作できる 「GitHub CLI」のベータ版がリリースされました。常時コマンドラインから離れられない（？）開発者にはピッタリかもしれません。

Docker Desktop 2.2正式版が登場。WSL 2対応をテクニカルプレビューとして。ファイルシステムもSambaからgRPC FUSEへ － Publickey(2020/2/3)

Winodws上でもHyper-Vを利用せずに、WSL 2のLinux環境上でDockerを直接実行できるようになった点が素晴らしいです。

アプリケーションにおける権限設計の課題 - kenfdev’s blog(2020/1/13)

アプリケーションの権限管理の設計は非常に難解ですが、こちらの記事では権限周りのベストプラクティスが紹介されており、非常に参考になりました。

働き方

スケールする組織を支えるドキュメンテーションの技術を”GitLab Handbook”から学ぶ｜Anno Takahiro｜note(2020/2/14)

1000人規模（！）でフルリモートワークをしているGitLab。ドキュメンテーションの文化や原則が非常に参考になります。

クラウド業界

国内ITサービス市場予測を発表 from IDC(2020/2/17)

ITシステムの「2025 年の崖」が話題になっているように、デジタルトランスフォーメーション（DX）の市場が2024年まで成長し続けるという予測が発表されました。

今後も面白く役に立つクラウド業界ニュースをお届けします。
